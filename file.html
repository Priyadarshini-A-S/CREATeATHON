Introduction to C Programming

What is C?

C is a powerful programming language developed by Dennis Ritchie in the early 1970's.
It is widely used for system programming, operating systems and embedded applications

Features of C

  • Simple and Efficient
  • Structured programming language
  • Portable and Platform-Independent
  • Low level access to memory
  • Rich library Support

Basic Syntax

Here is an example of a simple c program :

                
                #include<stdio.h>
                int main(){
                    printf("Hello World!\n");
                    return 0;
                    }

Applications of C

  • System Softwares(eg.:Operating Systems)
  • Embedded systems
  • Game Development
  • Compilers and Interpretors

C Programming Syntax

General structure of a C Program

Every C Program follows a basic structure as shown below :

/*Include Header Files*/
#include<header_file>

/*Main Function*/
int main() {
/* Code logic */
return 0;
}

Example C Program

A simple "Hello, World!" program in C:

#include <stdio.h> /* Include standard input-output header */

return 0; /* Return 0 to indicate successful execution */
}

Key Components of the Syntax

  • Header Files: Include necessary libraries using #include.
  • Comments: Add notes or descriptions using /* */ for multi-line comments or // for single-line comments.
  • Main Function: The program execution starts with the main() function.
  • Code Block: Code is enclosed in curly braces { }.
  • Statements: Each statement ends with a semicolon ;.
