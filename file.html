Introduction to C Programming

What is C?

C is a powerful programming language developed by Dennis Ritchie in the early 1970's.
It is widely used for system programming, operating systems and embedded applications

Features of C

  • Simple and Efficient
  • Structured programming language
  • Portable and Platform-Independent
  • Low level access to memory
  • Rich library Support

C Programming Syntax

General structure of a C Program

Every C Program follows a basic structure as shown below :

/*Include Header Files*/
#include<header_file>

/*Main Function*/
int main() {
/* Code logic */
return 0;
}

Example C Program

A simple "Hello, World!" program in C:

#include <stdio.h> /* Include standard input-output header */

return 0; /* Return 0 to indicate successful execution */
}

Key Components of the Syntax

  • Header Files: Include necessary libraries using #include.
  • Comments: Add notes or descriptions using /* */ for multi-line comments or // for single-line comments.
  • Main Function: The program execution starts with the main() function.
  • Code Block: Code is enclosed in curly braces { }.
  • Statements: Each statement ends with a semicolon ;.

Variables in C

What is a Variable>

A Variable in C is a container that holds data during the executio of the program.Variables must be declared before they can be used.

Syntax for declaring a Variable :

For Example:

int age;/*Declares an integer variable */
float salary;/*Declares a floating point variable*/

Types of Variables

  • int : Used for integers (for eg.:5,78,-89,etc...).
  • float : Used for floating-point numbers(for eg.:5.6,7.89,etc).
  • char : Used for single characters(for eg.:'f','G').
  • double : Used for double-precision floating-point numbers.

Example Pragram:

#include<stdio.h> /*Include Standard Input-Output Header*/
int main(){
int age=25; /*Integer Variable */
float salary=45000.00; /*Float point Variable*/ char grade='A'; /*Character Variable*/
printf("Age : %d\\n",age); /*To Print the Integer*/
printf("Salary : %.2f\\n",salary); /*To Print the Float*/
printf("Grade : %c\\n",grade); /*To Print the character*/

return 0; /*End of the Program*/
}

Rules for Naming Variables in C

  • Variable names must start with a letter or an underscore(_).
  • Variable names cannot have spaces or any specil characters.
  • Variable names are case-sensitive(for eg.:age and Age are different from each other.
  • Reserved keywords (like int,float)cannot be used as variable names.

Functions in C

What is a Function?

A Function in C is a block of code designed to perform a specific task. It helps organize and reuse code efficiently.

Syntax of a Function

return_type function_name(parameters){
//code logic
return value; //Optional and depends on the return type
}

Types of Functions

  • built-in Functions :Functions provided by C(for eg,:printf,scanf).
  • User-Defined Functions : Functions created by the user that is the programmer to perform a specific tasks.

Example for a User-Defined Functions

This Program calculates the sum of two numbers using a function :

#include<stdio.h> /*Include standard input-output Header */

/*Function Declaration*/
int addNumbers(int a,int b);

int main(){
int num1=10,num2=20;
int sum=addNumbers(num1,num2); /*Function call */
printf("Sum:%d\\n",sum);
return 0;
}

/* Function Declaration */
int addNumbers(int a,int b){
return a+b;
}

Benefits of Using Functions

  • Reusability : Write a function once and can be used multiple times.
  • Modularity : Break down large programs into smaller,manageable pieces.
  • Readaptability : Makes the program easier to read and debug.
  • Code Maintainence :Easier to modify or update specific parts of the program.

Key Points

  • A Function must be declared before it is called, either via a prototype or directly before the main() function.
  • The return type defines the type of the value the function returns(for eg.:int,float,void.

Loops in C

What is Loop?

A Loop in C is used to execute a block a code repeatedly until a specified condition is met.

Types of Loops

  • for loop : Executes a block of code a specified number of times.
  • While Loop : Repeats as long as a condition is true.
  • do-while loop : Executes a block of code atleast once, then continues if the continues if the condition is true.

1.for loop

the for loop is used when the number of iterations is known.

for(initialazation;condition;update){
}

Example :

#include<stdio.h>

int main(){
for(int i=1;i<=5;i++){
printf("%d\\n",i);
}
return 0;
}

2.While loop

The while loop is used when the condition is checked before executing the code block.

while(condition){
//code to execute
}

Example :

#include<stdio.h>

int main(){
int i=1;
while(i<=5){
printf("%d\\n",i);
i++;
}
return 0;
}

3.do-while loop

The do-whileloop ensures the code block is executed atleast once.

do{
//code to execute
}while(condition);

Example :

#include<stdio.h>

int main(){
int i=1;
do{
printf("%d\\n",i);
i++;
}while(i<=5);
return 0;
}

Benefits of Loops

  • Efficiency :Reduces code redundancy by automating repetitive tasks.
  • Flexibility : Handles Dynamic conditions during runtime.
  • Code ReadabilityImproves the Structure of the program.

Key Points

  • Ensure the loop has a termination condition to avoid infinite loops.
  • Use the break statement to exit a loop prematurely.
  • Use the continue statement to skip the current iteration.

Arrays in C

What is an Array?

An array in C is a collection of variables of the same type, stored in contiguous memory locations.

Syntax of an Array

data_type array_name[size];

Example:

int numbers[5]; /* An array of 5 integers */
float grades[10]; /* An array of 10 floating-point numbers */

Types of Arrays

  • One-dimensional Arrays: Used to store a list of elements.
  • Multi-dimensional Arrays: Used to store data in a tabular or matrix format (e.g., 2D arrays).

Example: One-dimensional Array

This program calculates the sum of elements in an array:

#include <stdio.h>

int main() {
int numbers[5] = {1, 2, 3, 4, 5};
int sum = 0;

for (int i = 0; i < 5; i++) {
sum += numbers[i];
}

printf("Sum of array elements: %d\\n", sum);
return 0;
}

Example: Two-dimensional Array

This program prints a 2x3 matrix:

#include <stdio.h>

int main() {
int matrix[2][3] = {
{1, 2, 3},
{4, 5, 6}
};

for (int i = 0; i < 2; i++) {
for (int j = 0; j < 3; j++) {
printf("%d ", matrix[i][j]);
}
printf("\\n");
}
return 0;
}

Benefits of Arrays

  • Efficient Data Storage: Allows storage of multiple elements of the same type.
  • Easy Access: Elements can be accessed using an index.
  • Memory Management: Stores data in contiguous memory locations.

Key Points

  • Array indices start from 0 in C.
  • The size of the array must be defined during declaration or initialization.
  • Out-of-bounds access (accessing elements outside the array size) can lead to undefined behavior.
